About the Project

Cleteci have a lot of time working with Seguros Pirámide, and have complete web system for the Agents of this important Insurance Company. Trhough the system the agents, hospitals and drugstores can manage important funcionalities reducing time and cost for all.

This was my introduction project on Cleteci. It was a short project where implement a Download Portal, to let the users access specific files of the server and with no order trought a clean interface as a file explorer in most operative systems.
